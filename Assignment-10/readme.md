# Assignment-10 : LeetCode Array Challenge

## Objective

To implement commonly asked array operations from coding interviews and LeetCode problems using C programming.

---

## Problem Statement

The program performs:

- Largest Element
- Second Largest Element
- Smallest Element
- Second Smallest Element
- Reverse Array
- Remove Duplicate Elements
- Frequency Count
- Searching
- Sorting
- Time Complexity Analysis

---

## Algorithm

1. Accept array elements from the user.
2. Perform the selected array operation.
3. Display the output.
4. Repeat until the user exits.

---

## Time Complexity

| Operation         | Complexity |
| ----------------- | ---------- |
| Largest           | O(n)       |
| Second Largest    | O(n)       |
| Smallest          | O(n)       |
| Second Smallest   | O(n)       |
| Reverse           | O(n)       |
| Remove Duplicates | O(n²)      |
| Frequency Count   | O(n²)      |
| Linear Search     | O(n)       |
| Bubble Sort       | O(n²)      |

---

## Space Complexity

O(n)

---

## Sample Input

5

20
10
40
30
20

---

## Sample Output

Largest Element = 40

Second Largest = 30

Smallest Element = 10

Second Smallest = 20

Array Sorted Successfully.

Element Found at Index 2

Duplicates Removed Successfully.
