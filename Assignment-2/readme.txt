# Assignment-02 : String Utility Toolkit

## Objective

To implement various string operations using arrays, loops and functions in C.

---

## Problem Statement

Develop a menu-driven program that performs:
- Length of String
- Reverse String
- Palindrome Checking
- Count Vowels
- Count Digits
- Count Spaces
- Character Frequency Count

---

## Algorithm

1. Accept a string from the user.
2. Display menu options.
3. Perform operations using functions.
4. Repeat until the user exits.
5. Display the required output.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Length | O(n) |
| Reverse | O(n) |
| Palindrome | O(n) |
| Counting | O(n) |
| Frequency Count | O(n) |

---

## Space Complexity

O(n)

---

## Sample Input

Hello World 123

---

## Sample Output

Length = 15

Reversed String:
321 dlroW olleH

Vowels = 3

Digits = 3

Spaces = 2

Character Frequency:
H = 1
e = 1
l = 3
o = 2
1 = 1
2 = 1
3 = 1
