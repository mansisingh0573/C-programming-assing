# Assignment-06 : Inventory Management System

## Objective

To develop an Inventory Management System using structures, arrays, searching and sorting techniques in C.

---

## Problem Statement

The program should:

- Add Products.
- Display Product Details.
- Search Products using Product ID.
- Update Product Information.
- Sort Products by Price.
- Calculate Total Inventory Value.

---

## Algorithm

1. Create a structure for storing product details.
2. Store records using an array of structures.
3. Display menu options.
4. Perform the selected operation.
5. Repeat until the user exits.

---

## Time Complexity

| Operation        | Complexity |
| ---------------- | ---------- |
| Add Product      | O(1)       |
| Display Products | O(n)       |
| Search Product   | O(n)       |
| Update Product   | O(n)       |
| Sort Products    | O(n²)      |
| Inventory Value  | O(n)       |

---

## Space Complexity

O(n)
