# Assignment-04 : Student Record Management System

## Objective

To develop a Student Management System using structures, arrays and functions in C programming.

---

## Problem Statement

The program should:

- Add Student Records.
- Display Student Records.
- Search Student by ID.
- Update Student Information.
- Delete Student Records.

---

## Algorithm

1. Create a structure for storing student information.
2. Display menu options.
3. Perform the selected operation.
4. Store records using an array of structures.
5. Repeat until the user exits.

---

## Time Complexity

| Operation | Complexity |
| --------- | ---------- |
| Add       | O(1)       |
| Display   | O(n)       |
| Search    | O(n)       |
| Update    | O(n)       |
| Delete    | O(n)       |

---

## Space Complexity

O(n)
