# Assignment-03 : Pointer Playground

## Objective

To understand the concept of pointers, memory addresses, dereferencing, pointer arithmetic and array traversal in C programming.

---

## Problem Statement

Create a program that demonstrates:

- Memory Address Representation
- Dereferencing Pointers
- Swapping Numbers using Pointers
- Traversing Arrays using Pointers
- Pointer Arithmetic

---

## Algorithm

1. Display menu options.
2. Accept user input.
3. Perform the selected pointer operation.
4. Display the result.
5. Repeat until the user exits.

---

## Time Complexity

| Operation          | Complexity |
| ------------------ | ---------- |
| Dereferencing      | O(1)       |
| Swapping           | O(1)       |
| Array Traversal    | O(n)       |
| Pointer Arithmetic | O(n)       |

---

## Space Complexity

O(n)

---

## Sample Input

Choice : 2

10
20

Choice : 3

5

1 2 3 4 5

---

## Sample Output

Before Swapping

a = 10
b = 20

After Swapping

a = 20
b = 10

Array Elements

1 2 3 4 5

---

## Learning Outcome

- Pointers
- Memory Management
- Pointer Arithmetic
- Arrays and Pointers
- Function Calling using Pointers
- Menu Driven Programming
