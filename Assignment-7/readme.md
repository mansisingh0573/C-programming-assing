# Assignment-07 : Hospital Patient Management System

## Objective

To develop a Hospital Patient Management System using structures, arrays and functions in C programming.

---

## Problem Statement

The program should:

- Register Patients.
- Display Patient Records.
- Search Patients using Patient ID.
- Update Patient Information.
- Delete Patient Records.

---

## Algorithm

1. Create a structure for storing patient details.
2. Store records using an array of structures.
3. Display menu options.
4. Perform the selected operation.
5. Repeat until the user exits.

---

## Time Complexity

| Operation        | Complexity |
| ---------------- | ---------- |
| Register Patient | O(1)       |
| Display Patients | O(n)       |
| Search Patient   | O(n)       |
| Update Patient   | O(n)       |
| Delete Patient   | O(n)       |

---

## Space Complexity

O(n)
