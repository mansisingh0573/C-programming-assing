# Assignment-08 : Employee Payroll System

## Objective

To develop an Employee Payroll System using structures and functions in C programming.

---

## Problem Statement

The program should:

- Accept employee details.
- Calculate HRA.
- Calculate DA.
- Calculate Tax.
- Calculate Gross Salary.
- Calculate Net Salary.
- Display Payroll Reports.
- Sort Employees based on Salary.

---

## Algorithm

1. Accept employee details.
2. Calculate salary components.
3. Store records using structures.
4. Display payroll reports.
5. Search employees using Employee ID.
6. Sort employees based on Net Salary.
7. Repeat until the user exits.

---

## Time Complexity

| Operation          | Complexity |
| ------------------ | ---------- |
| Add Employee       | O(1)       |
| Display Report     | O(n)       |
| Search Employee    | O(n)       |
| Sort Employees     | O(n²)      |
| Salary Calculation | O(1)       |

---

## Space Complexity

O(n)
