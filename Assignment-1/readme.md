# Assignment-01 : Student Marks Analyzer

## Objective

To develop a menu-driven C program for storing and analyzing student marks using arrays, functions, searching, and sorting techniques.

---

## Problem Statement

Store marks of N students and perform the following operations:

- Display marks.
- Find highest marks.
- Find lowest marks.
- Find total marks.
- Find average marks.
- Search marks.
- Sort marks in ascending order.

---

## Algorithm

1. Accept number of students.
2. Store marks in an array.
3. Display menu options.
4. Perform selected operations using functions.
5. Repeat until the user exits.

---

## Time Complexity

| Operation | Complexity |
| --------- | ---------- |
| Display   | O(n)       |
| Highest   | O(n)       |
| Lowest    | O(n)       |
| Total     | O(n)       |
| Average   | O(n)       |
| Search    | O(n)       |
| Sorting   | O(n²)      |

---

## Space Complexity

O(n)
